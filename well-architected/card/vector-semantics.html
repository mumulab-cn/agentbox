🎯核心概念

向量化是将文本、词汇转换为数值向量的过程，使计算机能够理解和处理语义信息。

文本 → 数值向量 → 语义计算
"苹果" → [0.2, -0.1, 0.8, ...] → 相似度计算

通过向量运算实现语义理解和推理。

📊Word2Vec

通过预测上下文词汇学习词向量表示：

CBOW: 上下文 → 中心词
Skip-gram: 中心词 → 上下文
P(w_o|w_i) = exp(v'_{w_o}^T v_{w_i}) / Σ exp(v'_{w_j}^T v_{w_i})

优势：训练快速，捕获语义关系好

🌐GloVe

结合全局统计信息和局部上下文：

J = Σ f(X_ij)(w_i^T w_j + b_i + b_j - log X_ij)²
X_ij = 词i和词j的共现次数

特点：利用全局词频统计，性能稳定

🤖BERT嵌入

基于Transformer的上下文相关表示：

h_i = Transformer(w_1, w_2, ..., w_n)[i]
上下文相关：同一词在不同句子中向量不同

优势：动态表示，语义理解更准确

⚡FastText

考虑子词信息的词向量：

v_w = Σ v_g (g ∈ G_w)
G_w = 词w的所有n-gram子串集合

优势：处理未登录词，支持形态变化

📐余弦相似度

最常用的向量相似度度量方法：

cos(θ) = (A · B) / (||A|| × ||B||)
= Σ(A_i × B_i) / (√Σ(A_i²) × √Σ(B_i²))

范围：[-1, 1]，1表示完全相似，-1表示完全相反

优势：不受向量长度影响，只关注方向

📏欧几里得距离

计算向量间的直线距离：

d(A,B) = √Σ(A_i - B_i)²
相似度 = 1 / (1 + d(A,B))

特点：距离越小，相似度越高

应用：聚类分析，最近邻搜索

🎲曼哈顿距离

计算向量各维度差值的绝对值之和：

d(A,B) = Σ|A_i - B_i|
也称为L1距离或城市街区距离

特点：对异常值不敏感

应用：推荐系统，图像处理

🔍点积相似度

直接计算向量的点积：

sim(A,B) = A · B = Σ(A_i × B_i)
未归一化的相似度度量

特点：考虑向量的大小和方向

应用：注意力机制，神经网络

相似度计算实现

import numpy as np from sklearn.metrics.pairwise import cosine_similarity from scipy.spatial.distance import euclidean, manhattan # 示例向量 vector_a = np.array([1, 2, 3, 4, 5]) vector_b = np.array([2, 3, 4, 5, 6]) vector_c = np.array([1, 1, 1, 1, 1]) # 余弦相似度 def cosine_sim(a, b): return np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b)) # 计算各种相似度 cos_sim_ab = cosine_sim(vector_a, vector_b) cos_sim_ac = cosine_sim(vector_a, vector_c) euclidean_ab = euclidean(vector_a, vector_b) manhattan_ab = manhattan(vector_a, vector_b) print(f"向量A和B的余弦相似度: {cos_sim_ab:.4f}") print(f"向量A和C的余弦相似度: {cos_sim_ac:.4f}") print(f"向量A和B的欧几里得距离: {euclidean_ab:.4f}") print(f"向量A和B的曼哈顿距离: {manhattan_ab:.4f}") # 批量计算相似度矩阵 vectors = np.array([vector_a, vector_b, vector_c]) similarity_matrix = cosine_similarity(vectors) print("相似度矩阵:") print(similarity_matrix) # 语义搜索示例 def semantic_search(query_vector, document_vectors, top_k=3): """语义搜索函数""" similarities = cosine_similarity([query_vector], document_vectors)[0] top_indices = np.argsort(similarities)[::-1][:top_k] results = [] for idx in top_indices: results.append({ 'index': idx, 'similarity': similarities[idx] }) return results # 模拟文档向量 doc_vectors = np.random.rand(10, 100) # 10个文档，每个100维 query_vec = np.random.rand(100) # 查询向量 search_results = semantic_search(query_vec, doc_vectors) print("搜索结果:") for result in search_results: print(f"文档{result['index']}: 相似度 {result['similarity']:.4f}")
