Flask 示例应用

这是一个使用 Flask 框架构建的完整示例应用，展示了现代 Web 开发的最佳实践

本应用集成了用户管理、博客系统、搜索功能和 RESTful API， 采用响应式设计，支持多种数据操作和交互功能。

核心功能
用户管理系统

完整的用户 CRUD 操作，包括用户列表、详情查看、添加新用户等功能

博客系统

支持文章发布、分类管理、多视图展示，提供丰富的内容管理功能

智能搜索

全文搜索功能，支持用户和文章的多维度搜索，提供搜索建议

RESTful API

完整的 API 接口，支持 JSON 数据交换，便于前后端分离开发
