配置说明

1. 创建飞书应用

访问 飞书开放平台 创建企业自建应用

2. 获取应用凭证

在应用管理页面获取 App ID 和 App Secret

3. 配置权限

为应用添加以下权限：

  • 获取与发送单聊、群组消息
  • 读取通讯录
  • 获取群组信息

4. 发布应用

将应用发布到企业内部使用
